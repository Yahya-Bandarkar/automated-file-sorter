Frequently Asked Questions

Find answers to common questions about Auto File Sorter

Is Auto File Sorter safe to use?

Absolutely! Auto File Sorter never modifies or deletes your original files unless you explicitly tell it to. All file operations are logged and can be undone at any time. We take data safety very seriously.

What operating systems are supported?

Auto File Sorter currently runs on Windows 10/11 and macOS 10.15+. Linux support is planned for a future release. The system requirements are minimal - it will run on any modern computer.

Can I try before I buy?

Yes! The Basic version is completely free forever with no limitations on usage. The Pro version offers a 14-day free trial with all features enabled so you can test it with your specific files before purchasing.

How does the undo functionality work?

Auto File Sorter maintains a detailed log of all file operations. When you undo, it uses this log to move files back to their original locations. You can undo multiple operations in sequence, one at a time.

What file types are supported?

Auto File Sorter supports over 100 file types out of the box, including documents (PDF, DOCX, TXT), images (JPG, PNG, GIF), videos (MP4, MOV, AVI), audio files (MP3, WAV), archives (ZIP, RAR), and more. You can also add custom file types.
